CSS attr() 函數三欄示範

attr(title)

滑鼠移上我看看 

<p>
  <span title="我是來自 title 屬性的文字">
    滑鼠移上我看看
  </span>
</p>

span::after {
    content: "（" attr(title) "）";
    color: blue;
}
/* 顯示元素的 title 屬性內容 */

attr(data-info)

商品 A
商品 B
 
<div data-info="這是自訂 data 屬性">商品 A</div>
<div data-info="限量發售中">商品 B</div>

div::after {
    content: " → " attr(data-info);
    color: green;
}
/* 顯示 data-info 自訂屬性值 */

attr(href)

Example 網站 
<a href="https://example.com">
  Example 網站
</a>

a::after {
    content: " (" attr(href) ")";
    color: gray;
}
/* 自動顯示超連結的 href */
